### 合成画像

| ファイル名 | 説明 |
|-----------|------|
| `07_composite.jpg` | 3枚の画像を地上基準で合成した画像 |

**合成方法**:
1. 地上の赤マーカー5点を使ってホモグラフィ変換行列を計算
2. 2枚目・3枚目を1枚目基準に変換
3. 3枚を不透明度1/3ずつにして加算合成
4. これにより地上が揃い、星の動きが軌跡として表現される

## 観察のポイント

1. **日周運動の方向**: オリオン座は東（左下）から西（右上）へ移動
2. **移動速度**: 1時間40分で約25°移動（1時間あたり約15°= 地球の自転速度）
3. **実測と理論の一致**: コンパス計測値と天文計算の理論値が3°以内で一致
